üêç 1. Installing Python & Flask

Step 1: Get Python

Flask needs Python. Let's install it:

  1. Go to python.org in your browser

  2. Click the big yellow button that says "Download Python 3.x.x"

  3. When the file downloads, DOUBLE-CLICK it to open (don't right-click!)

  4. CRITICAL: On the installer screen, CHECK THIS BOX:

    If you miss this, nothing will work later. It's at the bottom!

  5. Click "Install Now" ‚Üí Enter admin password if asked ‚Üí Wait for "Setup was successful"

Step 2: Install Flask

Now we install Flask using the command line:

  1. Open your computer's "Terminal" (Mac) or "Command Prompt" (Windows)

    How to find it:

    • Windows: Type "cmd" in Start Menu search
    • Mac: Open Spotlight (Cmd+Space) and type "Terminal"

  2. Type this EXACTLY and press Enter:

    pip install flask

    You should see "Successfully installed flask..."

  3. Verify it worked by typing:

    flask --version

    You should see something like: Flask 2.0.1

If you see "pip is not recognized":

  1. You missed the "Add to PATH" step during Python install
  2. Uninstall Python ‚Üí Reinstall ‚Üí CHECK "Add to PATH"!

üì¶ 2. Setting Up Virtual Environment

This creates a "clean room" for your project (prevents package conflicts):

  1. Create a folder for your project:

    • Windows: Right-click desktop ‚Üí New ‚Üí Folder ‚Üí Name it todo-app
    • Mac: Open Finder ‚Üí Desktop ‚Üí File ‚Üí New Folder ‚Üí Name it todo-app

  2. Open Terminal/Command Prompt and go to your folder:

    cd Desktop/todo-app

    If your folder is somewhere else, replace "Desktop/todo-app" with your path

  3. Create the virtual environment:

    python -m venv venv

    Nothing will happen if it works (that's good!)

  4. Activate it (this makes your terminal use the virtual environment):

    For Windows:

    venv\Scripts\activate

    For Mac/Linux:

    source venv/bin/activate

    You should see (venv) at the start of your terminal line

üê¨ 3. Setting Up MySQL

Step 1: Install MySQL

  1. Download MySQL Installer:

    • Go to mysql.com/downloads/installer
    • Click "MySQL Installer for Windows" (or macOS if you're on Mac)

  2. Run the installer (double-click the downloaded file)

  3. Select "Developer Default" ‚Üí Click "Next"

  4. When asked to "Check Requirements", click "Execute" for each item

  5. On the "Accounts and Roles" screen:

    • Set root password to: password123 (yes, really!)
    • Confirm it twice

  6. Click "Next" through everything else ‚Üí "Execute" ‚Üí "Finish"

Step 2: Verify MySQL Works

  1. Open a NEW Terminal/Command Prompt (important!)

  2. Type this to log into MySQL:

    mysql -u root -p

    When asked for password, type: password123 (you won't see typing)

  3. If you see this, you're IN:

    mysql>

  4. Type exit and press Enter to leave MySQL

If you get "mysql command not found":

  1. Reopen the MySQL Installer
  2. Go to "Products" ‚Üí Check "MySQL Server" ‚Üí Click "Reconfigure"
  3. On "Advanced Options", CHECK "Add MySQL to PATH"

üîå 4. Connecting Flask to MySQL

Step 1: Install Database Connector

In your TERMINAL (with (venv) active):

pip install flask-mysql

Step 2: Create config.py File

In your todo-app folder, create a new file named config.py

# config.py
MYSQL_HOST = 'localhost'
MYSQL_USER = 'root'
MYSQL_PASSWORD = 'password123'
MYSQL_DB = 'todo_db'

Important: This file contains your database password! Never share it.

üóÉÔ∏è 5. Creating Database Tables

Step 1: Create Database in MySQL

Open Terminal and log into MySQL:

mysql -u root -p

Step 2: Run These Commands in MySQL

Type each command below, then press Enter after each one:

CREATE DATABASE todo_db;
USE todo_db;
CREATE TABLE tasks (
id INT AUTO_INCREMENT PRIMARY KEY,
content VARCHAR(255) NOT NULL,
completed BOOLEAN DEFAULT 0
);

If you see "Query OK", it worked! Type exit to leave MySQL.

üöÄ 6. Writing the App Code

Step 1: Create app.py File

In your todo-app folder, create app.py with this code:

# app.py
from flask import Flask, render_template, request, redirect
from config import MYSQL_HOST, MYSQL_USER, MYSQL_PASSWORD, MYSQL_DB
from flask_mysqldb import MySQL

app = Flask(name)

Configure MySQL
app.config['MYSQL_HOST'] = MYSQL_HOST
app.config['MYSQL_USER'] = MYSQL_USER
app.config['MYSQL_PASSWORD'] = MYSQL_PASSWORD
app.config['MYSQL_DB'] = MYSQL_DB
mysql = MySQL(app)

Home page - show all tasks
@app.route('/')
def index():
cur = mysql.connection.cursor()
cur.execute("SELECT * FROM tasks")
tasks = cur.fetchall()
cur.close()
return render_template('index.html', tasks=tasks)

Add new task
@app.route('/add', methods=['POST'])
def add_task():
task = request.form['content']
cur = mysql.connection.cursor()
cur.execute("INSERT INTO tasks (content) VALUES (%s)", (task,))
mysql.connection.commit()
cur.close()
return redirect('/')

Delete task
@app.route('/delete/int:id ')
def delete_task(id):
cur = mysql.connection.cursor()
cur.execute("DELETE FROM tasks WHERE id = %s", (id,))
mysql.connection.commit()
cur.close()
return redirect('/')

if name == 'main':
app.run(debug=True)

Step 2: Create HTML Template

Create folder templates inside todo-app

Inside it, create index.html with this code:

<!DOCTYPE html>

</form>

<!-- Show all tasks -->
<div>
{% for task in tasks %}
<div class="task">
{{ task[1] }}
<a href="/delete/{{ task[0] }}">‚ùå</a>
</div>
{% endfor %}
</div>
</body>
</html>

‚òÅÔ∏è 8. Deployment (Baby-Level)

Running Locally Forever

To keep your app running when you close the terminal:

  1. Open Terminal ‚Üí Go to your project folder

  2. Activate virtual environment:

    venv\Scripts\activate # Windows
source venv/bin/activate # Mac

  3. Run in background (Windows):

    start /B python app.py

  4. Your app will keep running even if you close the terminal!

For real hosting (next steps after this guide):

  • Try Render.com (free tier)
  • They have a "Deploy Flask App" button that guides you
  • You'll need to create a "requirements.txt" file (we can teach this later!)
